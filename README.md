# Note App

> Simple todo application

## Технологический стек

- React
- React Router
- React Hooks
- Axios

## Стили

- SASS

## Разработка

### Установка зависимостей

```bash
yarn install
```

### Запуск dev версии

```bash
yarn start
```

## Производство

### Построение сборки

```bash
yarn install && yarn build
```
